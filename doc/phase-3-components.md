# Phase 3: Component Engineering

## Objective
Build the functional and aesthetic sections of the site.

## Components Breakdown

### 1. Navigation & Layout
- **Navbar:** Sticky, glassmorphism effect (backdrop-blur), semantic navigation links.
- **Footer:** Copyright, social icons, minimal layout.

### 2. The Hero (`hero.tsx`)
- High-impact typography.
- "Research-lab" feel with a clean subtitle.
- Minimalist CTA buttons with high-contrast borders.

### 3. Grid Sections (Projects & Agents)
- **ProjectCard:** Title, description, tech stack tags, and links.
- **AgentCard:** Focus on "Architecture" with placeholder SVGs or clean boxes to represent system flow.
- Implementation of hover-induced "glow" or border shifts.

### 4. Technical Sections
- **TechStack:** A clean grid of monochrome icons with tooltips.
- **Research:** Markdown-style layout to simulate a "Lab Note" or paper abstract.

## Verification
- Mobile responsiveness for all cards and grids.
- Hover states and active navigation links functional.
